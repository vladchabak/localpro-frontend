# LocalPro Mobile

Flutter marketplace for local services (Airbnb-style) — providers list on map, clients search/chat.

- **Repo:** Flutter 3.41.9 + Dart
- **Backend:** Java 21 + Spring Boot 3.3.6
- **Backend URL:** `http://10.0.2.2:8080` (Android) / `http://localhost:8080` (Chrome)
- **Auth:** `Authorization: Bearer dev-token` (dev mode)
